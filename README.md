## ? О проекте

ANISTREAM - это современная одностраничная платформа для просмотра аниме, разработанная для пользователей, которые ценят простоту и удобство. Проект реализован на чистом HTML, CSS и JavaScript без использования фреймворков.

## ✨ Особенности

### ? Продвинутый видеоплеер
- Кастомный плеер с интерфейсом в стиле YouTube
- Полноэкранный режим
- Регулировка громкости
- Изменение скорости воспроизведения (0.5x, 1x, 1.5x, 2x)
- Прогресс-бар с перемоткой
- Управление с клавиатуры

### ? Современный дизайн
- Темная тема в стиле медиа-сервисов
- Плавные анимации и переходы
- Адаптивный дизайн для всех устройств
- Эффекты при наведении
- Кастомные скроллбары

### ? Умный поиск и фильтрация
- Поиск по названию, оригинальному названию и жанрам
- Фильтрация по жанрам (Экшен, Драма, Комедия и др.)
- Динамическая загрузка контента
- Сохранение избранного в localStorage

### ? Полная адаптивность
- Оптимизация для компьютеров, планшетов и смартфонов
- Гибкая сетка карточек
- Адаптивный видеоплеер
- Удобная навигация на мобильных устройствах

## ? Управление

### Управление видеоплеером

| Клавиша | Действие |
|---------|---------|
| Пробел / K | Воспроизведение/пауза |
| F | Полноэкранный режим |
| M | Включить/выключить звук |
| Стрелки ← → | Перемотка на 10 секунд |
| Стрелки ↑ ↓ | Регулировка громкости |
| 0 / Home | В начало видео |
| End | В конец видео |
| > / . | Увеличить скорость |
| < / , | Уменьшить скорость |
| Escape | Выйти из полноэкранного режима |

### Мышь и тач

- Клик по карточке - Открыть аниме в плеере
- Клик по прогресс-бару - Перемотка видео
- Наведение на видео - Показать контролы
- Свайп на мобильных - Прокрутка контента

## ? Данные об аниме

Проект использует статические данные с информацией о 12 популярных аниме. Каждое аниме содержит:

```javascript
{
  id: 1,
  title: "Название на русском",
  originalTitle: "Оригинальное название",
  description: "Подробное описание",
  year: 2023,
  episodes: 24,
  rating: 9.0,
  genres: ["Жанр1", "Жанр2"],
  poster: "URL постера",
  videoUrl: "URL видео",
  featured: true // Главное аниме недели
}
```

## ? Настройка и кастомизация

### Изменение данных

Для добавления новых аниме отредактируйте файл data.js:

```javascript
{
  id: 13, // Уникальный ID
  title: "Новое аниме",
  // ... остальные поля
}
```

### Изменение стилей

Основные цвета и настройки в style.css:

```css
:root {
  --primary-color: #ff4757;    /* Основной цвет */
  --dark-bg: #0f0f23;          /* Фон */
  --text-primary: #ffffff;     /* Основной текст */
  /* ... другие переменные */
}
```

### Добавление видео

Замените ссылки на видео в data.js:

```javascript
videoUrl: "https://ваша-ссылка-на-видео.mp4"
```

## ? Отладка и логирование

```javascript
console.log('Действие пользователя:', действие);
```
